# Authentication Module

## Mục tiêu

Xây dựng hệ thống xác thực người dùng.

## Chức năng

* Đăng ký tài khoản
* Đăng nhập
* Đăng xuất
* Quản lý Session
* Phân quyền User/Admin

## File dự kiến

* login.php
* register.php
* logout.php
* profile.php

## Bảng dữ liệu liên quan

users

## Validation

* Username không được để trống
* Email đúng định dạng
* Password tối thiểu 8 ký tự

## Kết quả mong đợi

Người dùng có thể đăng ký, đăng nhập và đăng xuất thành công.
Admin và User được chuyển đến giao diện phù hợp sau khi đăng nhập.
