# Movie Management Module

## Mục tiêu

Quản lý thông tin phim trong hệ thống.

## Chức năng

* Thêm phim
* Sửa phim
* Xóa phim
* Xem danh sách phim
* Tìm kiếm phim
* Lọc phim theo thể loại
* Upload poster phim

## File dự kiến

* movies.php
* add_movie.php
* edit_movie.php
* delete_movie.php

## Bảng dữ liệu liên quan

movies
categories

## Validation

* Tên phim không được để trống
* Thời lượng phải lớn hơn 0
* Poster đúng định dạng ảnh

## Kết quả mong đợi

Admin có thể quản lý toàn bộ danh sách phim.
Người dùng có thể xem và tìm kiếm phim.
