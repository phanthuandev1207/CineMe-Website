# Booking & Admin Module

## Booking Module

### Chức năng

* Chọn suất chiếu
* Chọn ghế
* Đặt vé
* Xem lịch sử đặt vé

### Bảng dữ liệu

showtimes
bookings

### Kết quả

Người dùng có thể đặt vé thành công và lưu thông tin vào database.

---

## Admin Module

### Chức năng

* Dashboard
* Quản lý người dùng
* Quản lý vé
* Thống kê số lượng vé bán

### File dự kiến

* admin.php
* dashboard.php
* manage_users.php
* manage_bookings.php

### Bảng dữ liệu

users
bookings
movies

### Kết quả

Admin có thể quản lý toàn bộ hệ thống từ một trang điều khiển.
